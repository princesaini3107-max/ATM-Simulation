# ATM Simulation System 

This is a Python-based ATM Simulation project developed using a menu-driven approach. The system allows users to perform basic banking operations in a simple and interactive way.

##  Features
- Check Account Balance
- Deposit Money
- Withdraw Money
- View Transaction History (Statement)

## Concepts Used
- Python Functions
- Classes & Objects (OOP)
- List (for transaction history)
- Modular Programming (separate files for logic)
- Menu-driven infinite loop

## Project Structure
ATM-Simulation/
│
├── main.py
├── models/
│     └── account.py
├── services/
│     └── atm_service.py

## How to Run
1. Open the project folder in terminal
2. Run the following command:
   python main.py
